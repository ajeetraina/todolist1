# A Simple Todo List


## Using Undocker


- Clone the repository
- Install the dependencies

```
 npm install
````

- Start Redis

```
 docker run -d -p 6379:6379 redislabs/redismod
```

- Start the app

```
 nodemon .
```





## Using Docker

- Docker Desktop 


## Bring up the application


```
docker-compose up -d
```

Open https://localhost:3000
